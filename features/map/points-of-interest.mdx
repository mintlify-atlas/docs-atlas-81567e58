# Points of interest

CUCEI Ubícate maps every major building, module, and facility on the CUCEI campus. Each point is a tappable area on the map that opens a bottom sheet with detailed information.

## Point data structure

Each point of interest is defined in `Src/Screens/Home/Components/MapComponent/data.js` with the following properties:

```javascript data.js
export const points = [
  {
    id: "Biblioteca",
    name: "Biblioteca",
    aliases: ["Biblioteca", "biblioteca", "becas", "internacionalizacion"],
    left: 540,   // x-coordinate on the 1600×1400 map image
    top: 1035,   // y-coordinate on the 1600×1400 map image
    height: 73,  // tap target height in px
    width: 80,   // tap target width in px
    rotate: 5,   // rotation angle in degrees
  },
  // ...
];
```

The `aliases` array enables flexible searching — any alias string will resolve to the correct point when you search in either the route or specific search bars.

## Tapping a point

When you tap any marked area on the SVG map, the `handlePointPress` function is called with the point's `id`. This triggers the `BottomSheetComponent` to expand with information about that specific location.

```javascript HomePage.js
const handlePointPress = (pointId) => {
  setSelectedPoint(pointId);
  setIsBottomSheetVisible(true);
  bottomSheetRef.current?.expand();
};
```

The bottom sheet snaps to 50% height by default and can be expanded to 75% by swiping up.
