# CUCEI Ubícate

**CUCEI Ubícate** is a mobile app built for students at the Centro Universitario de Ciencias Exactas e Ingenierías (CUCEI), Universidad de Guadalajara. It puts everything you need to navigate and thrive on campus right in your pocket.

## What you can do

- **Find your way** — Search for any building, classroom, module, or facility on the interactive campus map
- **Follow routes** — Activate turn-by-turn routes with embedded video walkthroughs
- **Ask the bot** — Get answers to campus questions 24/7 from the CUCEI AI Assistant
- **Manage your account** — Register, log in, and personalize your profile with a custom avatar
- **Access services** — Quickly reach info about scholarships, social service, medical services, and CID
- **Stay informed** — Read campus articles and browse the staff directory
- **Listen to CUCEI Radio** — Tune into the university radio stream from the app
- **View your curriculum** — Browse the malla curricular for your degree program

## Technology

CUCEI Ubícate is built with:

| Layer | Technology |
|---|---|
| Mobile framework | React Native 0.76 + Expo 52 |
| Navigation | React Navigation 6 (stack + drawer) |
| Backend | Custom REST API (`ilabtdi.com/api_ubicate`) |
| AI | Stack AI (chatbot inference) |
| Storage | AsyncStorage (local), Supabase (cloud) |
| Maps | Custom SVG campus map with react-native-svg |
| UI | React Native Paper + FontAwesome |
